###css3的子字符串匹配属性选择器

+ 以特定前缀开头
+ 包含特定字符串
+ 以特定后缀结尾

1. img[alt^="film"]{
     border:3px solid red;
    }
2. img[alt*="film"]{
     border:3px solid red;
    }
3. img[alt$="film"]{
     border:3px solid red;
    }
